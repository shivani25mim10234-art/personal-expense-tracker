# personal-expense-tracker

## Overview
The Personal Expense Tracker is a Python-based application that helps users track, manage, and analyze their daily expenses. It allows users to:
- Record expenses with name, amount, and category
- View all expenses in an organized format
- Calculate total spending
- Generate category-wise summary
- Delete unwanted or incorrect expenses

## Features
- Add new expense
- View all expenses
- Calculate total expenses
- Category-wise summary
- Delete expense
- Easy, interactive menu-driven interface

## Technologies Used
- Python 3.x
- Lists and dictionaries
- Command-line interface (CLI)

## Setup / How to Run
1. Clone the repository:
git clone https://github.com/your-username/personal-expense-tracker.git
2. Navigate to the project folder:
cd personal-expense-tracker
3. Run the program:
python expense_tracker.py

## How to Use
1. Run the program to see the menu with options 1–6:
   - Add Expense  
   - View Expenses  
   - Total Expenses  
   - Category Summary  
   - Delete Expense  
   - Exit
